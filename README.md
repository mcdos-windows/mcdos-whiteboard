# mcdos的白板
一款基于Pygame开发的全屏黑板风格涂鸦工具，支持多色画笔、粗细调整等功能，界面简洁，操作直观。

## 核心功能
- 全屏深绿黑板背景，线条永久保留不消失
-  多色画笔：白/红/黄/蓝/粉5种颜色可选
-  画笔粗细可调（2-15）
-  极简操作：鼠标绘制，按钮/快捷键控制
